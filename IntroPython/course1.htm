ITS 140 - Intro to Comp Algorithms (with Python)

This course will cover the fundamental concepts related to programming using the Python language.

Topics include:

  • python
  • variables
  • modules
  • algorithms
  • logic
  • conditional structures
  • iterative structures
  • lists
  • list comprehensions
