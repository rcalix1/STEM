ITS 520 - Applied Machine Learning

This course will cover the fundamental concepts related to machine learning. Topics include:

  • the basic machine learning pipeline
  • data sets
  • WEKA
  • performance metrics
  • optimization
  • data scraping and data pre-processing
  • features and the vector space model
  • the bag of words approach for text 
  • traditional machine learning algorithms
  • sklearn
  • Tensorflow 2.0
  • numpy arrays and tensor operations
  • features from images
  • keras
  • reinforcement learning
  • unsupervised learning (clustering)
  • unsupervised learning (compression with SVD)

Prerequisites: Programming skills up to data structures and a senior/graduate level course in statistics. Knowledge of Python and Linux.
