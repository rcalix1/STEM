Building an Antikythera with 3d printing

This blog post describes some parts of the process by which I built a work of art inspired by the antikythera mechanism using 3D printing.

Before I started, I knew nothing about the antikythera and very little about 3D printing.

What is the Antikythera

A type of clock found in greece and believed to have been build more than 2000 years ago. also, so call it the first computer.

What did I use to build it?

I used a 3D printer. I experimented with 2 printers as I was learning but found one to be the best for this and actully was really happy with it.

The 3D printer that I preferred for this build is the Kobra Max for Anycubic. This is the sponsored link which supports this content Kobra Max 3D printer

This printer is big but can print all parts. And I rreally like the head bed of it.
